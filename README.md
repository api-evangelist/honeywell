# Honeywell (honeywell)
Honeywell is a Fortune 500 company that provides developer APIs and integration capabilities through its developer platform. Honeywell provides developer APIs for building automation, industrial IoT, aerospace data, and connected building management. The Forge platform enables partners to integrate with Honeywell's industrial and commercial systems.
